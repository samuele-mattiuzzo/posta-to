What is Posta-to?

Posta-to is a simple blogging application designed specifically for comic stripes and short messages as well. It's based on djangoappengine and uses django-filetransfers for image uploading.

The project code is currently hosted on GitHub and it provides a script that downloads all the required packages. See the README file for further instructions.

What does it do?

  • Register a new user
  • Login user
  • View user profile (using googlecharts)
  • View user charts
  • Create a new post
  • Upload an image with a post
  • Upvote/downvote a post
  • Delete a post
  • Browse posts
  • View rankings

What doesn't?

  • Delete an image
  • Resize image on upload
  • Share the posts

What is planned?

  • Tagging system (maybe using django-tagging)
  • Image uploading with resize (templatetag generates too overhead)
  • Auth with OpenID
  • Post sharing
  • New css colors (yeah)
